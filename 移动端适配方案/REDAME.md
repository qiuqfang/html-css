# 移动端适配方案

## 使用 CSS 媒体查询@media

## 使用 rem 单位 + @media

将页面像素转换为 rem 单位，并且在不同的设备上自动调整字体大小，以适应不同的设备屏幕尺寸。通过`@media`查询，可以让不同的设备上的字体大小保持一致。

## 使用 rem 单位 + flexible

将页面像素转换为 rem 单位，并且在不同的设备上自动调整字体大小，以适应不同的设备屏幕尺寸。通过**flexible**，可以让不同的设备上的字体大小保持一致。

- px 自动转换 vw
